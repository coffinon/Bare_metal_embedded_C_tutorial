# Bare_metal_embedded_C_tutorial

Board -> Nucleo F103RB

IDE -> STM32CubeIDE (clear - without CubeMX addon and HAL or StdPeriph libraries)

I only use a few header files like :
- cmsis_gcc.h
- cmsis_compiler.h
- cmsis_version.h
- core_cm3.h
- stm32f103xb.h
- system_stm32f1xx.h

So only essentials - no HAL at all :)
